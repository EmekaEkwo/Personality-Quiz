# Personality Quiz

A simple JavaScript personality quiz: **“What Type of Footballer Are You?”**

- 4 questions
- 4 possible outcomes (Playmaker, Finisher, Engine, Defender)
- Built with HTML, CSS, Bootstrap, and vanilla JavaScript.
